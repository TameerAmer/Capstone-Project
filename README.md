# OptiVision: Advanced Screening for Early Detection of Eye Health Issues

Overview
OptiVision is a desktop application designed to provide accessible and user-friendly tools for the early detection of vision impairments. The application features a series of scientifically validated vision tests, including:

Visual Acuity Test: Measures the sharpness of your vision.
Color Vision Test: Identifies potential color blindness issues.
Contrast Sensitivity Test: Assesses your ability to distinguish between contrasts.
Field of Vision Test: Evaluates your peripheral vision.
Blur Detection Test: Detects clarity issues with your vision.
The app guides users through a series of tests, ensuring accurate results with an initial screen calibration process. Test results are stored securely, allowing users to track their vision health over time and seek professional medical advice when needed.

Features
User-Friendly Interface: Clean, intuitive interface designed to make vision tests easy to perform at home.
Test Storage: Stores user test results in a secure database, enabling users to track their vision health over time.
Screen Calibration: Ensures accurate vision test results by adjusting the screen to real-world objects like a credit card or ID.
Responsive Design: The application is optimized for different screen sizes and provides a smooth user experience.

Project Structure
Frontend: Python
Backend: Django for processing user data, managing the database, and handling user interactions.
Database: MySQL for storing user information and test results.
